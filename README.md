An LCD (Liquid Crystal Display) is an electronic display module commonly used in embedded systems and microcontroller projects to show text, numbers, and simple symbols. It is widely used with microcontrollers like Arduino because it is easy to interface and consumes very little power.Main Components

Display Screen – Shows the characters.
Controller Chip – Controls the data and display operations.
Backlight – Provides illumination for the display.
Pins – Used to connect the LCD with a microcontroller.

🔌 Basic Pin Configuration (16×2 LCD)
VSS – Ground
VDD – Power supply (5V)
V0 – Contrast adjustment
RS – Register select
RW – Read/Write

E – Enable
7–14 – Data pins
15 – LED+ (Backlight positive)
16 – LED− (Backlight negative)
